### Dark Mode Configuration [Source: architecture.md#9, front-end-spec.md#3.1]

- **Dark-mode-first:** Classe `dark` no `<html>` por padrao
- CSS variables pattern (shadcn/ui padrao):
  ```css
  :root { --background: 0 0% 100%; --foreground: 222.2 84% 4.9%; ... }
  .dark { --background: 222.2 84% 4.9%; --foreground: 210 40% 98%; ... }
  ```
- Tailwind v4 usa CSS variables nativamente

### Coding Standards [Source: architecture.md#16]

- **Absolute imports:** Sempre usar `@/` (ex: `@/components/ui/button`)
- **Server Components por padrao:** So usar `'use client'` quando necessario
- **Naming:** Files kebab-case, Components PascalCase, Hooks use-prefix
- **No `any`:** TypeScript strict, usar `unknown` + type guards
